# NextBotAdapter

[TShock](https://github.com/Pryaxis/TShock) 插件，用于适配 [NextBot](https://github.com/Arispex/nextbot)。

## 安装

1. 下载最新版本
2. 将 `NextBotAdapter.dll` 放入 TShock 的 `ServerPlugins/` 目录
3. 启动服务器 —— 配置文件将自动创建于 `tshock/NextBotAdapter/`
