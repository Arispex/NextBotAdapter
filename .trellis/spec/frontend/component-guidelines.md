# Component Guidelines

> How components are built in this project.

---

## Overview

This repository currently has **no UI component layer**.

There are no frontend component files, no JSX / TSX files, and no server-rendered template components. Because of that, there is no component convention to enforce yet.

---

## Component Structure

Not applicable in the current codebase.

If a frontend is added later, document the real file shape after components exist.

## Styling Patterns

## Accessibility

There is no browser UI in this repository today, so no frontend accessibility standard is currently encoded in the codebase.

---

## Common Mistakes

- Do not assume this repository already has React-style components.
- Do not create speculative component guidelines without real component code.
- If a management or admin UI is introduced later, write these rules from the actual implementation.
